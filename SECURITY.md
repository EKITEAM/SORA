# Security Policy

## Supported Versions

The latest stable release of SORA receives security updates and bug fixes.

| Version | Supported |
|----------|-----------|
| Latest Release | ✅ |
| Older Releases | ❌ |

## Reporting a Vulnerability

If you discover a security vulnerability in SORA, please report it responsibly.

Please do not disclose security issues publicly before they have been reviewed.

You can report vulnerabilities by opening a private security advisory on GitHub or by contacting the maintainer directly.

When reporting a vulnerability, please include:

- A description of the issue
- Steps to reproduce the problem
- Potential impact
- Any suggested fixes (if available)

All reports will be reviewed as quickly as possible. Valid security issues will be investigated and addressed in a future release.
