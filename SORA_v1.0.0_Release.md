# SORA v1.0.0

Initial public release of SORA, an Android image loading library for API 21+.

## Highlights

- Multi-level caching (memory + disk)
- Network loading with retries and timeouts
- SVG support
- Animated image support for GIF and animated WebP
- Smart resizing and downsampling
- Request cancellation
- Image load observation via `ImageLoadObserver`
- Placeholder and error drawables
- Android 5.0+ support

## Installation

```gradle
dependencies {
    implementation 'com.github.EKITEAM:SORA:v1.0.0'
}
```
